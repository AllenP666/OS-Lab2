# Лабораторная работа: Кроссплатформенная библиотека для запуска процессов

Этот проект содержит C++ библиотеку для запуска и управления дочерними процессами в фоновом режиме. Библиотека является кроссплатформенной и работает в ОС семейств Windows и UNIX (Linux, macOS).

В состав проекта входят:
* `process_lib` — сама статическая библиотека.
* `dummy_app` — тестовое консольное приложение, которое работает 5 секунд и завершается с кодом `42`.
* `test_utility` — утилита, которая использует `process_lib` для запуска `dummy_app` и проверки кода его завершения.
